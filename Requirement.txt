What should I install for InstaBot?

1. Python v3.9.x >>> 3.9.6

    from ==> https://www.python.org/downloads/


2. Selenium(latest version) >>> 21.2.4

    pip install selenium

    * MicrosoftEdgeDriver
    * FireFox Driver
    * Chrome Driver

    Download from ==> https://selenium-python.readthedocs.io/installation.html#drivers
